# 📊 Polynomial Regression on Real Dataset

## 🚀 Project Overview

In this project, I applied **Polynomial Regression** on a real-world dataset to understand how it performs compared to a simple Linear Regression model.

The goal was to explore how well these models capture patterns in real data.

---

## 🧠 What I Did

* Applied **Linear Regression** as a baseline model
* Applied **Polynomial Regression** to capture non-linear relationships
* Compared both models using evaluation metrics (R² Score, MSE)
* Visualized the results for better understanding

---

## 📉 Results & Observations

* Both Linear and Polynomial Regression models **did not perform well** on this dataset
* Increasing polynomial degree did not significantly improve performance
* This indicates that:

  * The dataset may not follow a simple or polynomial pattern
  * More complex models or feature engineering might be required

---

## 💡 Key Learnings

* Real-world data is often messy and does not follow ideal assumptions
* Model performance depends heavily on data characteristics
* Simply increasing model complexity does not guarantee better results
* Understanding failure is an important part of Machine Learning
